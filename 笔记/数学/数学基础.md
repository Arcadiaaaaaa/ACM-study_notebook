# 数学基础

## 1. 整数反转

```cpp
long long t = x, b = 0;
while (t) {
    b = b * 10 + (t % 10);
    t /= 10;
}
```

注意：
- 数据范围大时，要考虑反转过程中是否溢出。
- 若 `x = 0`，要注意循环是否还能得到正确结果。

## 2. 最大公约数 `gcd`

```cpp
long long gcd(long long a, long long b) {
    return b == 0 ? a : gcd(b, a % b);
}
```

## 3. 三角形判定

若三边为 `a, b, c`，构成三角形的条件是：
- `a > 0`，`b > 0`，`c > 0`
- 任意两边之和大于第三边

若已知 `a <= b <= c`，只需判断：

```cpp
a + b > c
```

注意：
- 数据范围大时，`a + b` 可能溢出，必要时使用 `long long`。

## 4. 模运算

下列运算中，可以边算边取模：
- 加法
- 减法
- 乘法

常用性质：

```cpp
(a + b) % mod = ((a % mod) + (b % mod)) % mod
(a - b) % mod = ((a % mod) - (b % mod) + mod) % mod
(a * b) % mod = ((a % mod) * (b % mod)) % mod
```

注意：
- 除法不能直接写成“先除再取模”。
- 模意义下做除法，通常要乘逆元，而且有前提条件。
- 即使最终会取模，中间过程也可能先溢出。

## 5. 取余、取模、负数修正

在 C++ 中，`%` 是取余，不是数学意义上的“总是非负”的取模。

例如：

```cpp
-2 % 26 == -2
```

因此常见修正写法是：

```cpp
(x % m + m) % m
```

作用：
- 把结果规范到 `0 ~ m - 1` 范围内。

例子：

```cpp
(c - 'a' - n) % 26              // 可能错
((c - 'a' - n) % 26 + 26) % 26  // 更稳
```

## 6. 同余

如果

```cpp
a ≡ b (mod m)
c ≡ d (mod m)
```

那么有：

```cpp
a + c ≡ b + d (mod m)
a - c ≡ b - d (mod m)
a * c ≡ b * d (mod m)
```

结论：
- 加、减、乘都能把数先化到模意义下再计算。
- 这也是很多计数题、循环节题、密码题的基础。

## 7. 点到直线距离

若点 `A`、`B` 确定一条直线，点 `P` 到该直线的距离为：

```cpp
d = |AB x AP| / |AB|
```

二维向量叉乘：

```cpp
(x1, y1) x (x2, y2) = x1 * y2 - y1 * x2
```

注意：
- 分子要取绝对值。
- 这是“点到直线”的距离，不是“点到线段”的距离。
