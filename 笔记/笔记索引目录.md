# 笔记索引目录

这份索引不是按文件名堆目录，而是按“遇到什么问题该看哪里”来组织。

使用建议：
- 已经知道关键词时，先看“按问题 / 关键词查找”
- 只记得大概在哪一类时，再看“按文件查找”
- 还是找不到时，直接全文搜索：`rg -n "关键词" 笔记`
