# 字符串

## 1. `find`

作用：查找子串第一次出现的位置。

```cpp
string s = "aaBOBcc";
size_t pos = s.find("BOB");
```

说明：
- 找到则返回起始下标。
- 找不到则返回 `string::npos`。
- 区分大小写。

## 2. `substr`

作用：截取子串。

```cpp
string s = "abcdef";
cout << s.substr(2, 3);  // cde
cout << s.substr(2);     // cdef
```

说明：
- `substr(pos, len)`：从 `pos` 开始，截取长度为 `len` 的子串。
- `substr(pos)`：不写 `len` 时，从 `pos` 开始一直截取到字符串末尾。

## 3. `compare`

作用：比较字符串或子串。

说明：
- 返回 `0` 表示相等。
- 小于 `0` 表示当前串字典序更小。
- 大于 `0` 表示当前串字典序更大。

注意：
- `compare` 同样区分大小写。

## 4. `size_t`

说明：
- `size_t` 常用于表示字符串长度、数组大小、下标等。
- 它是无符号整数类型。

注意：
- 和 `int` 混用时要注意比较问题。
- 手写循环时，能用 `i < s.size()` 就不要写成 `i <= s.size()`。

## 5. 统一大小写

```cpp
for (char &c : s) c = tolower(c);
```

用途：
- 忽略大小写比较
- 文本匹配前的预处理

## 6. 按单词匹配，不要直接拿 `find` 顶替

如果题目要求匹配“完整单词”，`find` 往往不够，因为它查的是子串。

例如要查单词 `he`：
- 在 `"the"` 中，`find("he")` 会找到。
- 但这不代表单词 `he` 单独出现过。

这类题更稳的做法是：
- 先统一大小写
- 再按空格切出一个个单词
- 最后逐个比较

## 7. 字符串常用 STL

### 7.1 `reverse`

```cpp
reverse(s.begin(), s.end());
```

作用：
- 翻转字符串

### 7.2 `sort`

```cpp
sort(s.begin(), s.end());
```

作用：
- 按字符升序排序

### 7.3 `erase`

```cpp
s.erase(pos, len);
```

作用：
- 从 `pos` 开始删除长度为 `len` 的子串

### 7.4 `push_back` 和 `pop_back`

```cpp
s.push_back(c);
s.pop_back();
```

作用：
- 在末尾添加 / 删除一个字符

## 8. 字符串与整数互转

字符串转整数：

```cpp
int x = stoi(s);
long long y = stoll(s);
```

整数转字符串：

```cpp
string s = to_string(x);
```

手写转整数：

```cpp
int x = 0;
for (char c : s) x = x * 10 + (c - '0');
```

字符数字互转：

```cpp
int x = c - '0';
char c = char(x + '0');
```

## 9. 做题提醒

- `'7'` 是字符，`7` 是整数
- 手写转整数别忘了 `c - '0'`
- `stoi/stoll` 只适合范围装得下的时候
- 只想处理每一位时，直接按字符处理通常更稳
