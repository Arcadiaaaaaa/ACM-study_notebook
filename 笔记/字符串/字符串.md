# 字符串

## 1. `find`

作用：查找子串第一次出现的位置。

```cpp
string s = "aaBOBcc";
size_t pos = s.find("BOB");
```

说明：
- 找到则返回起始下标。
- 找不到则返回 `string::npos`。
- 区分大小写。

## 2. `substr`

作用：截取子串。

```cpp
string s = "abcdef";
cout << s.substr(2, 3);  // cde
```

说明：
- `substr(pos, len)`：从 `pos` 开始，截取长度为 `len` 的子串。

## 3. `compare`

作用：比较字符串或子串。

说明：
- 返回 `0` 表示相等。
- 小于 `0` 表示当前串字典序更小。
- 大于 `0` 表示当前串字典序更大。

注意：
- `compare` 同样区分大小写。

## 4. `size_t`

说明：
- `size_t` 常用于表示字符串长度、数组大小、下标等。
- 它是无符号整数类型。

注意：
- 和 `int` 混用时要注意比较问题。
- 手写循环时，能用 `i < s.size()` 就不要写成 `i <= s.size()`。

## 5. 统一大小写

```cpp
for (char &c : s) c = tolower(c);
```

用途：
- 忽略大小写比较
- 文本匹配前的预处理

## 6. 按单词匹配，不要直接拿 `find` 顶替

如果题目要求匹配“完整单词”，`find` 往往不够，因为它查的是子串。

例如要查单词 `he`：
- 在 `"the"` 中，`find("he")` 会找到。
- 但这不代表单词 `he` 单独出现过。

这类题更稳的做法是：
- 先统一大小写
- 再按空格切出一个个单词
- 最后逐个比较
