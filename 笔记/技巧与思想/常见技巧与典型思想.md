# 常见技巧与典型思想

## 1. 方向数组

作用：把网格中“相邻位置”的变化量统一存起来，便于循环处理。

八方向：

```cpp
int dx[8] = {-1, -1, -1, 0, 0, 1, 1, 1};
int dy[8] = {-1, 0, 1, -1, 1, -1, 0, 1};
```

示例：

```cpp
for (int k = 0; k < 8; k++) {
    int nx = x + dx[k];
    int ny = y + dy[k];
    if (0 <= nx && nx < n && 0 <= ny && ny < m && isboom(nx, ny)) {
        cnt++;
    }
}
```

注意：
- 使用前先统一坐标含义。
- 越界判断要写全。

## 2. 规则表 / 映射表

适用场景：
- 题目规则本身就是一一对应关系。

示例：

```cpp
beat["elephant"] = "tiger";
if (beat.count(a) && beat[a] == b) {
    // a 能赢 b
}
```

注意：
- 如果 `a` 可能不存在，先判断 `count`，避免误插入默认值。

## 3. 编号后利用取余找规律

适用场景：
- 多个对象构成环形关系。

示例：

```cpp
// elephant = 0, tiger = 1, cat = 2, mouse = 3
// x 吃 y
y == (x + 1) % 4
```

核心思想：
- 先编号。
- 再把题目的循环关系转成模运算。

## 4. 约瑟夫问题

关键思想：
- 删除一个人后，对剩余元素重新编号。
- 把原问题转化为规模减一的同类问题。

本质：
- 下标映射
- 重编号
- 规模缩小
