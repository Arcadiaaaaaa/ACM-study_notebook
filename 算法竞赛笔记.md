# 算法竞赛笔记

## 1. 输入输出

### 1.1 C++ 流加速

作用：减少 `cin/cout` 与 `stdio` 同步带来的开销。

```cpp
ios::sync_with_stdio(false);
cin.tie(nullptr);
```

注意：
- 开了加速后，尽量不要和 `printf/scanf` 混用。

### 1.2 控制小数位数

头文件：

```cpp
#include <iomanip>
```

示例：

```cpp
cout << fixed << setprecision(2) << x;
```

说明：
- `fixed`：固定小数形式输出。
- `setprecision(n)`：配合 `fixed` 时，表示保留 `n` 位小数。

### 1.3 控制宽度、对齐、填充

示例：

```cpp
cout << setw(5) << x;                  // 默认右对齐
cout << left << setw(5) << x;          // 左对齐
cout << setfill('0') << setw(5) << x;  // 不足补 0
```

说明：
- `setw(n)` 只对紧随其后的一次输出生效。
- `setfill(c)` 会持续生效，直到再次修改。

### 1.4 进制输出

示例：

```cpp
cout << dec << x;  // 十进制
cout << hex << x;  // 十六进制
cout << oct << x;  // 八进制
```

### 1.5 C 风格格式化输出

头文件：

```cpp
#include <cstdio>
```

示例：

```cpp
printf("%d %.2f\n", a, b);
```

常用格式符：
- `%d`：`int`
- `%lld`：`long long`
- `%.2f`：保留 2 位小数
- `%c`：`char`
- `%s`：字符串

### 1.6 读取字符串

示例：

```cpp
cin >> s;       // 读到空格为止
getline(cin, s); // 读取整行
```

注意：
- 如果前面刚用过 `cin` 读数字，再用 `getline`，要先处理掉残留换行。


## 2. 常用函数与语法

### 2.1 常用函数

```cpp
abs(x);                          // 绝对值
sqrt(x);                         // 平方根
reverse(s.begin(), s.end());     // 翻转
toupper(c);                      // 转大写
tolower(c);                      // 转小写
```

常见头文件：
- `abs`：`<cstdlib>` 或 `<cmath>`
- `sqrt`：`<cmath>`
- `reverse`：`<algorithm>`
- `toupper/tolower`：`<cctype>`

注意：
- `toupper`、`tolower` 处理的是单个字符。

### 2.2 开二维 `vector`

```cpp
vector<vector<int>> a(n, vector<int>(m));
```

### 2.3 三目运算符

示例：

```cpp
cout << (cond ? x : y);
```

说明：
- 三目运算符优先级较低，复杂表达式建议加括号。

### 2.4 范围 for

值遍历：

```cpp
for (char c : s) {
    // c 是副本
}
```

引用遍历：

```cpp
for (char &c : s) {
    // c 是原元素的引用
}
```

常量引用遍历：

```cpp
for (const auto &x : a) {
    // 只读，不拷贝
}
```

说明：
- 值遍历修改 `c` 不会影响原容器。
- 引用遍历修改 `c` 会影响原容器。

### 2.5 作用域

说明：
- 在一个函数内部定义的局部变量，其他函数不能直接访问。
- 如果其他函数要使用，通常有两种做法：
  - 作为参数传入。
  - 定义为全局变量。


## 3. 数学基础

### 3.1 整数反转

```cpp
long long t = x, b = 0;
while (t) {
    b = b * 10 + (t % 10);
    t /= 10;
}
```

注意：
- 如果题目数据范围大，要考虑反转过程中是否会溢出。

### 3.2 最大公约数 gcd
/
```cpp
long long gcd(long long a, long long b) {
    return b == 0 ? a : gcd(b, a % b);
}
```

### 3.3 三角形判定

若三边为 `a, b, c`，构成三角形的条件是：
- `a > 0`，`b > 0`，`c > 0`
- 任意两边之和大于第三边

若已知 `a <= b <= c`，只需判断：

```cpp
a + b > c
```

注意：
- 数据范围大时，`a + b` 可能溢出，必要时使用 `long long`。

### 3.4 模运算

下列运算中，可以边算边取模：
- 加法
- 减法
- 乘法

常用性质：

```cpp
(a + b) % mod = ((a % mod) + (b % mod)) % mod
(a - b) % mod = ((a % mod) - (b % mod) + mod) % mod
(a * b) % mod = ((a % mod) * (b % mod)) % mod
```

注意：
- 除法不能直接写成“先除再取模”。
- 模意义下做除法，通常要乘逆元，且需要额外条件。
- 即使最终会取模，中间过程也可能先溢出。

### 3.5 负数取模修正

在 C++ 中，负数取模结果可能仍为负数。

例如：

```cpp
-2 % 26 == -2
```

通用修正写法：

```cpp
(x % m + m) % m
```

作用：
- 把结果规范到 `0 ~ m - 1` 范围内。

例子：

```cpp
(c - 'a' - n) % 26          // 可能错
((c - 'a' - n) % 26 + 26) % 26
```

### 3.6 点到直线距离

若点 `A`、`B` 确定一条直线，点 `P` 到该直线的距离为：

```cpp
d = |AB x AP| / |AB|
```

二维向量叉乘：

```cpp
(x1, y1) x (x2, y2) = x1 * y2 - y1 * x2
```

注意：
- 分子要取绝对值。
- 这是“点到直线”的距离，不是“点到线段”的距离。


## 4. STL 容器

### 4.1 `stack`

头文件：

```cpp
#include <stack>
```

定义：

```cpp
stack<int> st;
```

常用操作：

```cpp
st.push(x);   // 入栈
st.pop();     // 出栈，无返回值
st.top();     // 访问栈顶
st.empty();   // 是否为空
st.size();    // 元素个数
```

注意：
- `stack` 是后进先出（LIFO）。
- 没有普通遍历接口。
- 调用 `top()` 和 `pop()` 前要先判空。

### 4.2 `set` 与 `unordered_set`

头文件：

```cpp
#include <set>
#include <unordered_set>
```

定义：

```cpp
set<int> s1;              // 有序、自动去重
unordered_set<int> s2;    // 无序、自动去重
```

常用操作：

```cpp
s.insert(x);
s.erase(x);
s.find(x);
s.count(x);
s.empty();
s.size();
s.clear();
```

区别：
- `set`：底层通常为红黑树，时间复杂度 `O(log n)`，元素有序。
- `unordered_set`：底层通常为哈希表，平均 `O(1)`，元素无序，最坏 `O(n)`。

注意：
- `set` 不支持随机访问。
- `set` 中元素值不能直接修改。
- `unordered_set` 存自定义类型时通常需要自定义哈希。

### 4.3 `map` 与 `unordered_map`

头文件：

```cpp
#include <map>
#include <unordered_map>
```

定义：

```cpp
map<string, int> mp1;
unordered_map<string, int> mp2;
```

常用操作：

```cpp
mp[key] = value;      // 插入或更新
mp.erase(key);        // 删除
mp.find(key);         // 查找
mp.count(key);        // 是否存在
mp.empty();           // 是否为空
mp.size();            // 元素个数
```

遍历：

```cpp
for (auto &p : mp) {
    cout << p.first << ' ' << p.second << '\n';
}
```

区别：
- `map`：有序，`O(log n)`
- `unordered_map`：无序，平均 `O(1)`，最坏 `O(n)`

重要注意：
- `mp[key]` 在 `key` 不存在时会自动插入默认值。
- 如果只是判断是否存在，优先用 `find` 或 `count`。
- 自定义类型作为 `unordered_map` 的键，通常需要自定义哈希。


## 5. 字符串

### 5.1 `find`

作用：查找子串第一次出现的位置。

示例：

```cpp
string s = "aaBOBcc";
size_t pos = s.find("BOB");
```

说明：
- 找到则返回起始下标。
- 找不到则返回 `string::npos`。
- 区分大小写。

### 5.2 `substr`

作用：截取子串。

示例：

```cpp
string s = "abcdef";
cout << s.substr(2, 3);  // cde
```

说明：
- `substr(pos, len)`：从 `pos` 开始，截取长度为 `len` 的子串。

### 5.3 `compare`

作用：比较字符串或子串。

说明：
- 返回 `0` 表示相等。
- 小于 `0` 表示当前串字典序更小。
- 大于 `0` 表示当前串字典序更大。

注意：
- `compare` 同样区分大小写。

### 5.4 `size_t`

说明：
- `size_t` 常用于表示字符串长度、数组大小、下标等。
- 它是无符号整数类型。


## 6. 常见技巧

### 6.1 方向数组

作用：把网格中“相邻位置”的变化量统一存起来，便于循环处理。

八方向：

```cpp
int dx[8] = {-1, -1, -1, 0, 0, 1, 1, 1};
int dy[8] = {-1, 0, 1, -1, 1, -1, 0, 1};
```

示例：

```cpp
for (int k = 0; k < 8; k++) {
    int nx = x + dx[k];
    int ny = y + dy[k];
    if (0 <= nx && nx < n && 0 <= ny && ny < m && isboom(nx, ny)) {
        cnt++;
    }
}
```

注意：
- 使用前先统一坐标含义。
- 越界判断要写全。

### 6.2 规则表 / 映射表

适用场景：
- 题目规则本身就是一一对应关系。

示例：

```cpp
beat["elephant"] = "tiger";
if (beat[a] == b) {
    // a 能赢 b
}
```

注意：
- 如果 `a` 可能不存在，先判断 `beat.count(a)`，避免误插入默认值。

### 6.3 编号后利用取余找规律

适用场景：
- 多个对象构成环形关系。

示例：

```cpp
// elephant = 0, tiger = 1, cat = 2, mouse = 3
// x 吃 y
y == (x + 1) % 4
```

核心思想：
- 先编号。
- 再把题目的循环关系转成模运算。


## 7. 典型思想

### 7.1 约瑟夫问题

关键思想：
- 删除一个人后，对剩余元素重新编号。
- 把原问题转化为规模减一的同类问题。

本质：
- 下标映射
- 重编号
- 规模缩小

这是建立递推式的核心。


## 8. 整理原则

### 8.1 检查结论是否有前提

例如：
- 模运算性质并不是对所有运算都成立。
- 几何公式要确认是否有绝对值、是否是直线还是线段。

### 8.2 检查 STL 操作是否有副作用

例如：
- `mp[key]` 会插入默认值。

### 8.3 检查模板是否匹配数据范围

例如：
- 是否会溢出。
- 是否要用 `long long`。
- 是否需要判空、判边界。

在向量v中间某一位置i之前插入变量x：
v.insert(v.begin() + i, x);
